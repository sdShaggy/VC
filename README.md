# Veridian Carbon — Production Website

A production-grade React + TypeScript + Tailwind CSS website for Veridian Carbon, featuring:

## Features

- **Storytelling Section** — Full-screen parallax scroll narrative
- **Horizontal Scroll Methodology** — Sticky scroll-driven horizontal card flow showing the 5-step carbon cycle
- **Animated Counters** — Intersection Observer-triggered number animations
- **Particle Canvas** — Lightweight canvas particle system on Hero
- **Scroll Reveal** — CSS-class based fade/rise reveal animations
- **Organic Shapes** — Morphing blob decorations
- **Grain Texture** — SVG noise overlay for organic feel
- **Fully Responsive** — Mobile-first design

## Typography

- **Display**: Playfair Display (serif) — headings
- **Body**: DM Sans — paragraph text
- **Mono**: JetBrains Mono — labels, badges

## Setup

```bash
npm install
npm run dev
```

Build for production:
```bash
npm run build
```

## Sections

1. **Hero** — Headline, CTA, particle background, stats pills
2. **Problem** — CDR gap + India vulnerability (with bar chart)
3. **Storytelling** — Scroll-driven narrative (4 chapters)
4. **Technology** — ERW + Biochar + Synergy cards
5. **Methodology** — Horizontal scroll carbon cycle (5 steps)
6. **dMRV** — Scientific rigor, 4-pillar grid
7. **Impact** — Animated counters, project map, roadmap
8. **Team** — 4-person grid
9. **Offtake** — Ocean bg CTA + LOI terms
